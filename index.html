Designed for everyone


Vast catalog of software with Flatpak

With Flatpak, you can install a vast software with a click of a button.

Visit Flathub

Simple web browser

Navigate to your favorite websites with GNOME Web, a simple and fast web browser.

Edit all your documents with OnlyOffice**

Edit your Word, Excel, PowerPoint and PDF documents with OnlyOffice.



Collaborate with your current video conferencing tools

Stay connected with your team by using your already existing video conferencing tools like Microsoft Teams, Zoom and more.**

Windows Compatibility*

Vincent OS is designed to be compatible with Windows software and games.
You can install Windows software with Wine / Proton (Steam only) / Lutris.

PowerShell by default

Vincent OS comes with PowerShell by default, unifying the power of PowerShell and the compatibility of bash.



Ship with ClamAV

To stay safe against malicious threads, Vincent OS ships with ClamAV.



Core LivePatch at your side

Core LivePatch is a in-house hotpatch system to applying critical and security updates without rebooting your computer.




Ready to Discover Vincent OS?

Actually, Vincent OS is still in development.
But you can download the Open Beta version of Vincent OS 1.0 and try it!
Don't hesitate to report bugs and issues on GitHub Issues.






* Some Windows applications may not run on Vincent OS. Refer to WineHQ - AppDB | ProtonDB and Lutris for compatibility information.
** OnlyOffice, Microsoft Teams, Zoom are not included on the Operating System. You need to install them manually.
